## Programming paradigm types: 2
1. Functional
2. Object Oriented

## Stateful vs stateless

### Stateful:
Can hold data. For example, Class.

### Stateless:
Can NOT hold data. For example, Functions.

## Class
- constructor
- keyword: this = inside the object, without this = outside the object
